== Description ==

EmailThis is a free Wordpress plugin which provides an easy way for visitors of your site to send your posts via email. It adds a small icon to every post so users can send the posts to their friends. People can share posts -> you will instantly get new readers. 


Please visit:

http://emailthis.seoblog.cz


== Installation ==

1. Download the plugin archive and unzip it.
2. Put it into wp-content/plugins/ directory.
3. Go to the WordPress Administration and create a new page with name "Forward this post" f.e. and pernament URL "http://www.your-site.com/email/". (important)
